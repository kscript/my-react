## react

### 安装
``` npm
npm i

``` 
### 打包/发布
``` npm
// 打包自己实现的react代码, 主要用于测试. 
// 如果是要发布, 可以直接执行 npm publish
npm run react
```

### 编译前端资源
``` npm
// 编译前端 + 抽取模块提供给浏览器
npm run source

// 抽取模块提供给浏览器
npm run browserify
```

### 编译并运行服务器
``` npm
npm run server
```

### 构建应用
``` npm
// 编译前端 + 运行并运行服务器
npm run build
```
